Assignments

Assignment 1: Basic HTML

Webpage about investing using lists, tables, images, and external links.

Assignment 2: Basic CSS

Webpage about Monte Carlo Simulation focusing on the styling. Using backgrounds, colors, padding and more!

Assignment 3: Page Layout

Webpage working on css skills, specifically skills with flexbox!

Assignment 4: Recreate CSS

Recreating a USC webpage utilizing html, and css flexbox! Also used different positioning skills and more!

Assignment 5: JavaScript Buttons, Functions, and More

First JavaScript assignment using what we learned in class to make text appear when clicked, being able to have a color picker embedded in the website, and changing images!

Assignment 6: Conditionals

Made a clock and a slider that displayed different images using JavaScript!

Assignment 7: Loops

Used loops to display CSS Art, and change the background based on the time of day!

Assignment 8: Arrays

Used associative arrays and loops to connect dog names to their images.

Assignment 9: Classes

Used JavaScript classes to display paintings with information like frames, the image, title, and creator
